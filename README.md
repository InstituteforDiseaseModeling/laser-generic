# LASER Generic

Generic disease models built with the LASER framework.

- SI/SIS
- SIR/SIRS
- SEIR/SEIRS

## Installation

Currently:

```bash
pip install git+https://github.com/InstituteforDiseaseModeling/laser-generic.git@v0.0.0
```

Once we have published on PyPI:

```bash
pip install laser-generic
```
